mill.

Turn sprint evidence into shareable artifacts.
PDF, CSV, Markdown, JSON-LD, static sites.

npx @grainulator/mill export --format pdf brief.html

Export

HTML and Markdown to PDF. Claims to CSV, JSON-LD, or clean Markdown.

Publish

Generate static sites from sprint outputs. Copy to clipboard for quick sharing.

Zero Deps

No installed dependencies. Heavy lifting via npx on demand.

Standalone

Works with any sprint output. No wheat installation required.

Usage

# Export brief to PDF
mill export --format pdf output/brief.html
# Export claims to CSV
mill export --format csv claims.json
# Convert HTML to Markdown
mill convert --from html --to markdown output/brief.html
# Build a static site
mill publish --target static output/
# Semantic web export
mill export --format json-ld claims.json -o claims.jsonld
